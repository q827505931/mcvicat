# 数据库管理工具

简洁的 MySQL 和 Redis 数据库管理桌面应用，类似 Navicat 的轻量级替代品。

## 功能

- 🔌 MySQL 连接管理
  - 查看数据库和表
  - 执行 SQL 查询
  - 表格形式展示结果

- 🔑 Redis 连接管理
  - Key 列表浏览
  - 查看和编辑值
  - 执行 Redis 命令

## 安装运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 打包
npm run build
npm run package
```

## 技术栈

- Electron
- React + TypeScript
- Tailwind CSS
- mysql2
- ioredis
